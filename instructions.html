Game Guide

How to Play

Follow the game journey: choose a mode, memorise the image, draw from memory, then beat your best score.

1

Choose a Mode

Select Easy, Normal, or Hard. Each mode changes how much time you get.

2

Memorise

Study the image carefully before it disappears. Focus on shape and position first.

3

Draw

Use the drawing tools to recreate the image from memory as accurately as possible.

4

Get Scored

Your drawing is compared with the original image to calculate your similarity score.

Best Strategy

Big shape first, details second.

Start by remembering the main outline, size, and position. Add smaller details only after the main shape is clear.

Difficulty Modes

Easy Longer time, simple images
Normal Balanced challenge
Hard Less time, harder images

Drawing Tools

Brush Eraser Line Rectangle Circle Undo Clear Grid

Score Tips

  • Remember the biggest shape first
  • Notice where the image sits on the canvas
  • Use simple shapes before small details
  • Keep your drawing clean and clear
